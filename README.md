## Project Overview

The Drive-Thru Restaurant Management System is a **modular, object-oriented Java application** designed for console-based interaction.  
It supports **admin and customer roles**, allowing for efficient management of restaurant operations.

- **Admin**: Add, edit, delete menu items, toggle availability, and view/update orders.
- **Customer**: Browse menu, manage cart, checkout with tax calculation, and view order history.

---

## Features

- User authentication (Admin / Customer)
- Admin menu management:
  - Add / Edit / Delete items
  - Toggle availability
- Customer functionalities:
  - Browse menu
  - Add / Update / Remove items in cart
  - Checkout with tax and payment method
  - View order history
- Simple and clean terminal UI
- In-memory order tracking
- Modular and maintainable OOP design
